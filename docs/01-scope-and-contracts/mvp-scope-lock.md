# MVP Scope Lock

## In Scope

- Money Health Score endpoint and dashboard display.
- FIRE Roadmap endpoint with Gemini API and deterministic fallback.
- 5-step onboarding UI with goal inputs.
- Mobile responsive interface for demo.

## Out of Scope (Hackathon MVP)

- Authentication and user accounts.
- Persistent portfolio tracking.
- Real brokerage or payment integrations.
- Full multilingual support beyond basic text readiness.

## Demo Success Criteria

- User can complete onboarding in one flow.
- Score and category breakdown are generated reliably.
- FIRE roadmap always returns valid output.
- App is live on public URL and works on mobile.
